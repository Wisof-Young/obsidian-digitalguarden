# SUMMARY

我们往往只看见一个人受人仰望，受人尊敬的一面。但是人是一个完整的个体，受人欢迎的一面来自于令人反感厌恶的另一面，二者不可分割。

当嫉妒一个人、想要成为一个人或者以某人为榜样的时候，想清楚，彼人身上的所有特点，自己是否能全部接受。
